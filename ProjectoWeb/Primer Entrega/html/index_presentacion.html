Nombre clave:

The Best Game Web

¿Que es?

The best Game Web sera un projecto que integre varias tecnologias dentro de una sola paguina web, nuestro reto sera presentar una paguina Web interactiva, con alta calidad visual (atractiva para la vista), con implementacion de almenos 3 juegos dentro de ella.

Objetivos Principal

  1. Desarrollar un website para juegos tipo GameWeb.

Objetivos secundarios:

  1. Implementacion tanto de CSS como de JavaScript para estilos dinamicos.
  2. Uso de otras tecnologias como lo es el Desarrollo.
  3. Desarrollar un login funcional.
  4. Tener un diseño limpio, intuitivo y agradable para el usuario.

Problema:

Se desea desarrollar un sitio web que pueda contener diferentes juegos tipo GameWeb, además, se requiere un login con el fin de ofrecer una mejor experiencia al usuario.

Prototipo:

En el desarrollo del prototipo inicial tendremos al menos 8 layers principales, dentro de estos estara una pantalla de Login, register, un Home en el momento de iniciar sesion, la galeria de juegos que te dirigira a la pantalla del juego, Un menu dentro de este y paguinas informativas explicando los controles (help) y el completo desarrollo del proyecto (about) & How work?.

Para el prototipo (diseño y Aplicacion) se uso la herramienta Figma. Una nueva herramienta que esta remplazando a Adobe XD en trabajos de diseño y que nos permite visualisar el prototipo en funcionamiento en Click Aca. o dandole click a la imagen:
